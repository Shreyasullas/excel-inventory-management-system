Inventory Management System using Excel VBA
# Project Overview

This project is an Inventory Management System built using Microsoft Excel and VBA macros.
It helps users manage products, track stock movement, monitor low stock items, and analyze inventory using dashboards and charts.

# Tools Used
Microsoft Excel
VBA (Visual Basic for Applications)
Excel Charts
Macros

# Features
Add and manage product information
Stock In / Stock Out transactions
Automatic stock update
Low stock alerts
Inventory summary report
Dashboard with charts
CSV dataset import
Transaction history tracking

Inventory_Project
│
├── Product_Master
├── Stock_Transactions
├── Inventory_Summary
├── Low_Stock_Report
├── Dashboard
├── DashboardData (Hidden Sheet)

# Dashboard Charts
The dashboard includes:
Stock Value by Category
Stock Quantity by Category
Number of Products by Category
Low Stock Products by Category

# 📂 Dataset
Dataset contains 100 products with fields:
Product ID
Product Name
Category
Price
Reorder Level
Current Stock

# ⚠ Low Stock Alert
The system automatically shows a warning message when stock reaches or falls below the reorder level.
